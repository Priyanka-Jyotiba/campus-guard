# Campus Guard - College Management System

## 📌 Overview
Campus Guard is a Django-based web application designed to manage college operations with role-based access for Admin, Faculty, and Students.

## 🚀 Features
- Role-based dashboards (Admin, Faculty, Student)
- Attendance management system
- Leave application system
- Student record management
- Secure authentication system

## 🛠 Tech Stack
- Python
- Django
- MySQL
- HTML, CSS, Bootstrap

## ⚙️ How to Run
1. Clone the repository:
   git clone https://github.com/Priyanka-Jyotiba/campus-guard.git

2. Navigate to project folder:
   cd campus-guard

3. Install dependencies:
   pip install -r requirements.txt

4. Run server:
   python manage.py runserver
